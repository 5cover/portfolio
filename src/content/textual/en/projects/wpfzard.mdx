<p>

<h3 id="result">Result</h3>
<p>
  In the end, I did not end up using **WPFZard** in WinClean. This is because I've found that previous versions of the
  Xceed toolkit are licensed under MIT, so I can legally use them in WinClean. It was a nice project, though, and I
  taught me a lot about making reusable WPF components.
